hello world

we make tools and games that encourage unique experiences with web browser technology that reject the modern assertion that most good things require a form of personal data harvesting and cloud storage.

the public library consists of javascript resources for offline behavior. this includes:

  • devices: motion, orientation, user media, and midi
  • lexicon: topics and constraints to be used as content to make games
  • music-detection: detecting music in audio streams
  • music-theory: music notation and key generation
  • procedural-generation: using current time and seeded rng to sync devices instead of a network
  • sound-synthesis: generating sound with web oscillators
  • steganography: encoding and decoding data in image pixels

games and interactive experiences include:

  • avva: two independent machines, one turns music into color, the other color into music. point them at each other, then get in-between.
  • lexicon-standoff: a 3+ player word game, try to determine who is an imposter
  • mindmeld: lightweight two person collaborative word game for any time
  • modulo: the coolest step sequencer with a command prompt
  • sonicpx: a music player, but images are the file format
